# Рекомендательная система

Мы создали универсальное решение, позволяющее использовать и обучать любые модели, которые отвечают двум условиям:

* обучаются с помощью метода .fit(X_train, y_train)
* делают предсказание с помощью метода .predict(data)

В качестве стандартной архитектуры предлагаем стекинг модель с тремя базовыми моделями и мета-моделью:
* RandomForestClassifier
* SVC
* XGBClassifier
* LogisticRegression - мета-модель

Также можно сохранять, загружать и дообучать различные модели
